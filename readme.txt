== Description ==

Wbcom Designs - BuddyPress Redirect plugin allows you to redirect users to different locations based on their user roles upon login and logout. 

For additional support or custom development needs, feel free to contact us at [Wbcom Designs](https://wbcomdesigns.com/contact/).

== Installation ==

1. Upload the entire "bp-redirect" folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How can we change redirection settings using this plugin? =

After installing the BP Redirect plugin, a new menu will appear in the WordPress admin panel. This menu links to the BP Redirect settings page, where you can configure all redirection settings.

= Does this plugin work only with the BuddyPress plugin? =

No, the BP Redirect plugin does not require BuddyPress to be active. It can work independently of BuddyPress.
